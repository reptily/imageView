# imageView

Run dev mode
```bash
# Clone this repository
git clone https://git@github.com:reptily/imageView.git
# Go into the repository
cd imageView
# Install dependencies
npm install
# Run the app
npm start
```

## Build for windows
```bash
# Install dependencies
npm install
# run building
electron-packager . imageView --platform=win32
```

## Build for Linux
```bash
# Install dependencies
npm install
# run building
electron-packager . imageView --platform=linux
```
